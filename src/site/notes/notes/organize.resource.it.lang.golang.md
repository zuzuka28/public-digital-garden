- `int64` зачастую можно заменить на `int`, тип `int` подстаивается по кол-ву бит под разрядность системы

- Менять значение `gomaxproc` имеет смысл если много системных вызовов, потому что они блокируют системные потоки, в которых исполняются горутины. При большЕм количестве тредов программа будет работать шустрее.
