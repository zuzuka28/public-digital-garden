<https://en.wikipedia.org/wiki/Total_sum_of_squares>


$$TSS =\sum _{i=1}^{n}\left(y_{i}-{\bar {y}}\right)^{2}$$

- $y_i$ - измерение $y$
- $\hat{y}$ - среднее всех измерений

## Матричная форма

$$TSS=(Y- \bar{Y})^T(Y-\bar{Y})$$
