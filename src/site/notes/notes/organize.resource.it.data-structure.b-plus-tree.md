B+ дерево - струтура данных, похожая на [[notes/organize.resource.it.data-structure.b-tree\|notes/organize.resource.it.data-structure.b-tree]], но при этом хранящая значения только на листах, не на промежуточных нодах.

Такие деревья лучше хранятся в памяти, чем B-деревья, поскольку содержит меньше ссылок на данные. 

B+ деревья можно просматривать последовательно.
